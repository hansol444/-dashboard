# Topic: 법무법인 비용 처리 (태평양 / 엘에이비파트너스)

## 개요
법무법인 두 곳에서 매월 자문료 청구 → Concur 송장 제출.
**세금계산서** 방식이므로 재무회계팀에 송장 담당자 변경 요청 후 연동 송장 처리.

| 법무법인 | 업무 성격 | 담당 변호사 |
|---------|----------|------------|
| 법무법인(유한) **태평양** | JEDI 법률자문 (큰 건) | 이수진 변호사, 곽한결 변호사 |
| 법무법인 **엘에이비파트너스(LAB)** | 월별 법률고문 (정기) | - |

> ⚠️ 예산품의/구매품의는 **창준님께 직접 요청**. Concur 처리만 인턴 담당.

## Concur 처리 절차 (두 법무법인 공통)

### Step 1. 청구서 수령 및 금액 확인
- ① 필요 서류: 청구서 (이메일로 수령)
- ② 입수처: 법무법인에서 창준님 CC 이메일로 발송 → 창준님 포워드 or 직접 수신
- ③ 확인: 창준님께 "금액 문제 없으신지 확인" 슬랙 후 OK 받으면 진행

### Step 2. 세금계산서 발행 요청
- ① 필요 서류: 청구서 확인 완료
- ② 입수처: 창준님께 "세금계산서 발행 부탁드려도 될까요?" 슬랙 → 법무법인 측에 요청 or 창준님이 직접 처리
- ③ 시스템: 세금계산서 수신 후 잡코리아 사업자번호 1108134859로 열람 → PDF 저장

### Step 3. Concur 처리
- ① 필요 서류: 예산품의 PDF, 세금명세서
  - ⚠️ 태평양의 경우: **명세서도 추가 필요** (2026-01 재무팀 확인)
  - 실비청구비가 있을 경우 별도 송장으로 따로 제출 필요
- ② 입수처: concursolutions.com (이메일 링크로 로그인)
- ③ 시스템 입력값:

**법무법인 태평양:**
| 필드 | 값 |
|------|-----|
| 공급업체 검색 | 법무법인(유한)태평양 |
| A-02. 송장 이름 | `법무법인(유한)태평양-JEDI 법률자문` |
| A-03~05. 날짜 | 세금명세서 기준 |
| D-01. 코스트센터 | 창준님 확인 (JEDI 관련) |
| 경비 유형 | 지급수수료-법률자문 (재무팀 확인) |

**법무법인 엘에이비파트너스(LAB):**
