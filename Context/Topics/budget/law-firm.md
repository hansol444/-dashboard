# Topic: 법무법인 비용 처리 (LAB 등)

## 개요
법무법인 LAB 등 법률 서비스 비용을 월별로 Concur 처리.

## 처리 주기
- 월 1회 정기 처리 (청구서 수령 후)

## Concur 처리 절차
1. 법무법인으로부터 청구서(세금계산서) 수령
2. Concur 접속 → Create Expense Report
3. 핵심 필드:
   - **Report Name**: 법무법인 LAB YYYYMM
   - **Business Purpose**: 법률 자문 서비스
   - **Vendor**: 법무법인 LAB
   - **Amount**: 청구 금액
4. 세금계산서 첨부
5. 재무회계팀 확인 후 제출

## 담당 연락처
- 법무팀 윤종화님/박민경님 — 법무 업무 내용 확인
- 재무회계팀 박은미님/왕윤형님 — Concur 처리
- 창준님 — 최종 승인
