# Topic: 기프티콘 구매·발송 & Concur

## 개요
KT alpha 플랫폼으로 네이버페이 기프티콘 대량 발송.
**두 케이스** 존재 — 전략실 자체 인터뷰 vs ATS 운영팀 인터뷰.
전체 흐름: 견적서 수령 → 예산품의 → 구매품의 → 집행신청서 발송 → 기프티콘 발송 → 수령자리스트 수령 → 거래명세서 수령 → Concur 작성

> **Concur 방식**: KT alpha는 세금계산서 없이 **인보이스(거래명세서)** 방식 → 새 송장 생성으로 직접 작성
> (세금계산서 방식 업체 예: 엔코라인, 엠브레인 → 해당 파일 참고)

## 케이스 A: 전략실 기프티콘 (산학협력·전략실 자체 인터뷰)

### Step 0-1. 대상자 리스트 수령
- ① 필요 서류: 이름 + 전화번호 리스트
- ② 입수처: 창준님 슬랙 DM
- ③ 시스템 입력값: -

### Step 0-2. KT alpha 담당자 변경 요청
- ① 필요 서류: -
- ② 입수처: 이메일 b2b.m@kt.com (G커머스운영팀 이덕희님), CC kang.sh@kt.com (강석현 대리님)
- ③ 시스템 입력값: 거래고객코드 **C000162772** (전략추진실 고유코드)

### Step 1. 견적서 수령 (KT alpha)
- ① 필요 서류: -
- ② 입수처: kang.sh@kt.com 강석현 대리님에게 이메일 문의
- ③ 시스템 입력값: -

### Step 2. 예산품의 작성
- ① 필요 서류: 견적서
- ② 입수처: 잡코리아 포털 → 전자결재 → 문건등록
- ③ 시스템 입력값:
  - 문건 제목: `[예산품의] 인터뷰 진행에 따른 기프티콘 구매를 위한 예산품의`
  - 읽기권한: 결재권자 + 동일팀
  - 문건분류: `[예산관리] 1.예산관리 (OPEX, CAPEX, MKT) - 5)예산 배정(신규, 전용, 초과) - 100만원 이하`
  - 시행일: 기프티콘 발송일
  - 코스트센터: **전략추진실**
  - G/L 계정: `[지급수수료 - 일반]` or `[접대비]` or `[마케팅비]` (창준님 확인)
  - 예산연동 유형: 대체

### Step 3. 구매품의 작성
- ① 필요 서류: 견적서, 예산품의 (하위문건)
- ② 입수처: 예산품의 페이지 → "하위문건 등록" 클릭
- ③ 시스템 입력값:
  - 문건 제목: `[구매품의] 인터뷰 진행에 따른 기프티콘 구매를 위한 구매품의`
  - 문건분류: `[주무부서 경우] > 구매 > 1) 100만원 미만`
  - 계약유무: 아니요
  - 공급사: KT alpha

> 품의 승인 완료 시 → 예산품의/구매품의 PDF 저장 (잡코리아 포털 → 전자결재 → MY문건 → 프린트 → PDF로 저장)

### Step 4-1. 집행신청서 발송 (KT alpha)
- ① 필요 서류: 집행신청서 (템플릿 복제 후 작성), 발송정보 시트
- ② 입수처: b2b.m@kt.com 이덕희님에게 이메일 발송. 희망 발송일 **1일 전**까지 접수
- ③ 시스템 입력값:
  - 집행신청서: 전략추진실 > 97. Concur 행정처리 자료 > ATS 인터뷰 비용 처리 > 집행신청서 템플릿
  - 상품코드/상품명/유효일수/판매가격: 기프티쇼 상품리스트 `<B2B 백화점 마트 주유>` 시트 참고

### Step 4-2. 수령자 리스트 수령
- ① 필요 서류: KT alpha에서 발송완료 내역리스트 수령
- ② 입수처: KT alpha 자동 발송
- ③ 시스템 입력값: 파일 비밀번호 = 잡코리아 사업자번호 **1108134859**
  → PDF 변환 전 Excel 민감도 설정 '공개'로 변경, 휴대폰 뒷 4자리 xxxx 처리

### Step 4-3. 거래명세서 수령
- ① 필요 서류: 거래명세서 (KT alpha 최예지님)
- ② 입수처: 수령자 리스트 수령 메일 회신으로 거래명세서 요청 → yeji.choi2@kt.com (회계팀 최예지님)
- ③ 시스템 입력값: '정산 내역 확인 요청의 건' 메일 확인 후 이상 없다고 회신

### Step 5. Concur 작성 (새 송장 생성)
> 마감: 매달 익월 D+4 영업일 오전

- ① 필요 서류 (5종 PDF): 예산품의, 구매품의, 견적서, 수령자 리스트, 거래명세서
- ② 입수처: concursolutions.com (이메일 링크로 로그인)
- ③ 시스템 입력값:

| 필드 | 값 |
|------|-----|
| A-01. 유형 | 일반 |
| A-02. 송장 이름 | ㈜케이티알파-기프티쇼 정산 |
| A-03. 증빙일 | 거래명세서 기준 날짜 |
| A-04. 업무완료일 | 기프티콘 전송한 날짜 |
| A-05. 전기일 | 거래명세서 기준 날짜 |
| A-06. 적요 | `케이티알파/날짜(25XXXX)/기프티쇼 구매/담당자 이름/비용항목` |
| B-01. 은행명 | (003) 기업은행 |
| B-02. 계좌번호 | (1000) 52500100697657 |
| C-01. 통화 | KRW-Korea(Republic), Won |
| C-02. 총 합계 | 거래명세서 금액 |
| C-04. 부가세 합계 | 0 |
| D-01. 코스트센터 | **(H2200) 전략추진실** |
| D-02. 지급 상태 | Not Paid |
| E-01. 세금코드 | (Z1) 매입-Invoice |
| E-02. 지급 방법 | 계좌이체-원화 |
| E-03. AP 조정계정 | A.06 2101-01 매입채무-일반 |

## 케이스 B: ATS 기프티콘 (ATS 운영팀 고객 인터뷰)

> 케이스 A와 절차 동일하나, **예산/코스트센터** 차이 있음.
> Concur 진행 전 명주형(ATS운영팀)님께 예산 집행 계획 사전 공유 필요.

### 다른 점 요약

| 항목 | 케이스 A (전략실) | 케이스 B (ATS) |
|------|----------|---------|
| 대상자 리스트 수령처 | 창준님 | ATS운영팀 **최돈민님** (슬랙 DM방) |
| 예산 코스트센터 | 전략추진실 | ATS운영팀(마케팅) 배부용 |
| G/L 계정 (예산품의) | 지급수수료-일반/접대비 | **광고선전비(콘텐츠)** |
| 받는 코스트센터 | 전략추진실 | **ATS사업실** |
| 받는 G/L 계정 | 지급수수료-일반/접대비 | **접대비-법인카드** |
| Concur 코스트센터 | (H2200) 전략추진실 | **(D2030) ATS사업실** |
| 사전 공유 필요 | 없음 | 명주형(ATS운영팀)님께 공유 |
