# Topic: 컨설팅 자문료 Concur 처리 (BCG)

## 개요
BCG(Boston Consulting Group)와의 Commercial Transformation 컨설팅 자문료 Concur 처리.
**착수금·중도금 등 대형 건** → 예산품의/구매품의는 창준님 직접 처리, PDF로 공유 후 Concur 작성.

> ⚠️ 2026-03-18 기준: 착수금 + 중도금 = **7.7억원** 규모
> 입찰생략요청서, 제안서, 계약서는 성욱님 보유.

---

## Concur 처리 절차

### [사전 준비] 파일 수집

| 파일명 | 기준 날짜 | 입수처 |
|--------|-----------|--------|
| 예산품의 PDF | - | 창준님께 PDF 공유 요청 |
| 구매품의 PDF | - | 창준님께 PDF 공유 요청 |
| 세금계산서 | 발행일 | BCG 측 발송 → 창준님 CC |
| 입찰생략요청서 | - | 성욱님 보유 |
| 제안서 | - | 성욱님 보유 |
| 계약서 | - | 성욱님 보유 |

> 예산품의/구매품의가 열람권한 없을 경우: 창준님께 "BCG 예산품의/구매품의 PDF 공유해 주실 수 있을까요?" 슬랙 요청

### Step 1. 재무회계팀 송장 담당자 변경 요청
- ① 필요 서류: -
- ② 입수처: 재무회계팀 **박은미님**에게 요청
- ③ 시스템: Outlook "송장이 지정됨" 메일 수신 → 본문 링크 클릭 → Concur 접속

### Step 2. 연동 송장 작성 (Concur)
> 마감: 매달 익월 D+4 영업일 오전

- ① 필요 서류: 예산품의 PDF, 구매품의 PDF, 세금계산서
- ② 입수처: concursolutions.com (이메일 링크로 로그인)
- ③ 시스템 입력값:

| 필드 | 값 |
|------|-----|
| A-01. 유형 | 일반 |
| A-02. 송장 이름 | `BCG-Commercial Transformation 자문료` (지급 유형에 따라 조정: 착수금/중도금/잔금) |
| A-03. 증빙일 | 세금계산서 기준 날짜 |
| A-04. 업무완료일 | 세금계산서 기준 날짜 |
| A-05. 전기일 | 세금계산서 기준 날짜 |
| A-06. 적요 | `BCG/날짜/Commercial Transformation 자문료/창준님` |
| D-01. 코스트센터 | **(H2200) 전략추진실** (창준님 확인) |
| D-02. 지급 상태 | Not Paid |
| E-01. 세금코드 | (V1) 매입세금공제-10%세금계산서-일반 |
| E-02. 지급방법 | 계좌이체-원화 |

> ⚠️ GL 계정/경비유형 사전 확인 필수 → 재무회계팀 박은미님·왕윤형님 Slack DM

### Step 3. 항목별 명세 요약
- H-01. 경비 유형: **지급수수료-컨설팅** (재무팀 확인 후 사용)
- H-02. 상세설명: `BCG Commercial Transformation 자문료 (착수금/중도금)` (지급 유형에 맞게 기입)
- H-03. 공급가액: 세금계산서 참고

### Step 4. 파일 첨부 & 제출
- 이미지 업로드: 예산품의, 구매품의, 세금계산서 첨부 → 저장
- 최종: "송장 제출"
