# Topic: 컨설팅 자문료 컨커 처리 (BCG 등)

## 개요
BCG 등 외부 컨설팅/자문 비용을 Concur로 처리.

## Concur 처리 절차
1. Concur 접속 → Create Expense Report
2. 핵심 필드:
   - **Report Name**: BCG 자문료 YYYYMM (예: BCG 자문료 202503)
   - **Business Purpose**: 경영 자문 서비스
   - **Vendor**: 해당 컨설팅사명 (예: Boston Consulting Group)
   - **Amount**: 청구 금액
   - **업무명**: 정확한 프로젝트명 기입 (자문 계약서 기준)
   - **비고**: 계약 기간, 자문 범위 등
3. 송장 + 계약서 첨부
4. 재무회계팀 담당자 변경 요청 후 제출

## 주의사항
- 컨설팅 자문료는 계약서 기반으로 비고란 상세 기입
- GL 계정 사전 확인 필수 (재무회계팀)

## 담당 연락처
- 재무회계팀 박은미님/왕윤형님 — 계정 확인, Concur 담당자 변경
- 창준님 — 자문 내용 및 금액 확인
