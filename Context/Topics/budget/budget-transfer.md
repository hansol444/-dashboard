# Topic: 추가 예산 품의 / 예산 이월

## 개요
기존 예산 한도 초과 또는 다음 기간으로 이월이 필요할 때 진행하는 추가 품의.
반려 확률이 높으므로 사유를 명확히 작성하는 것이 핵심.

## 추가 예산 품의 (예산 초과)

### 언제 필요한가
- 기존 예산품의 한도를 초과해서 집행해야 할 때
- 신규 비용 항목이 발생했을 때

### 프로세스
1. 잡코리아 포탈 → 전자결재 → 문건등록
2. 문서분류: [예산관리] 예산배정(신규, 전용, 초과)
3. **증액 목적과 사유를 구체적으로 작성** (반려 방지)
   - 예: "당초 예산 대비 OO 사유로 XX만원 초과 발생"
4. 첨부: 변경된 견적서, 사유서

### 주의사항
- 반려 확률 높음 → 창준님과 사전 얼라인 필수
- 사유 불명확 시 즉시 반려됨

## 예산 이월

### 언제 필요한가
- 당해 연도에 집행 못한 예산을 다음 연도로 넘길 때

### 프로세스
- 경영기획팀 임장식님께 문의 → 이월 가능 여부 및 절차 안내받기

## 담당 연락처
- 경영기획팀 임장식님 — 예산 이월 및 추가 품의 전반
- 창준님 — 사전 얼라인
