# Topic: Placement Survey 컨커 처리

## 개요
Placement Survey 관련 비용(엠브레인 용역비 등)을 예산품의 → 구매품의 → Concur 순서로 처리.

## 1단계: 예산품의
- 잡코리아 포탈 → 전자결재 → 문건등록
- 문서분류: 금액에 따라 1,000만원 이하/초과 선택
- 첨부: 견적서

## 2단계: 구매품의
- 예산품의 승인 후 → 하위문건 등록 클릭
- 문서분류: [구매] 해당 금액 구간
- 첨부: 견적서, 계약서 (구매검토 완료분)

## 3단계: Concur 처리
- Concur 접속 → Create Expense Report
- 핵심 필드:
  - **Report Name**: Placement Survey YYYYMM (예: Placement Survey 202503)
  - **Business Purpose**: Placement Survey 용역비
  - **Vendor**: 엠브레인
  - **Amount**: 견적서 금액
  - **GL Account**: 재무회계팀 확인 필요
- 송장(인보이스) 첨부 필수
- 담당자 변경 필요 시: 재무회계팀에 요청

## 담당 연락처
- 엠브레인 문주원님 — 견적서/인보이스 수령
- 재무회계팀 박은미님/왕윤형님 — GL 계정 확인, Concur 담당자 변경
- 창준님 — 최종 승인
