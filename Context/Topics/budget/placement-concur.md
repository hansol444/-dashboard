# Topic: Placement Survey 비용 처리 (예산품의 → 구매품의 → Concur)

## 개요
엠브레인이 분기별 Placement Survey 수행 후 용역비 청구.
전략추진실에서 25.3Q부터 직접 진행.
**26년 연간 예산품의/구매품의 완료** → 현재는 Concur 작업만 필요.
(27년 예산품의·구매품의 필요 시 창준님/총무팀/재무회계팀과 협의)

> **Concur 방식**: 엠브레인은 **세금명세서** 방식 → 재무회계팀에 담당자 변경 요청 후 연동 송장 처리
> (직접 새 송장 생성 X)

---

## Phase 1: 예산품의 (연 1회, 보통 전년도 말~연초)

> 26년 예산품의 이미 완료 (작성자: 주호연). 27년 준비 시 아래 참고.

### Step 1. 예산품의 작성
- ① 필요 서류: 엠브레인 견적서, 전년도 Placement Survey 장표 (아웃풋)
- ② 입수처: 창준님 → 엠브레인 CC 메일 또는 슬랙 / 이전 예산품의 문건 참고
- ③ 시스템 입력값 (잡코리아 포털 → 전자결재 → 문건등록):
  - 문서분류: `[예산관리] 1.예산관리 (OPEX, CAPEX, MKT) - 3)예산 배정(신규, 전용, 초과) - 1,000만원 초과`
  - 읽기권한: 결재권자 + 동일팀
  - 코스트센터: **(M2010) 잡코리아 마케팅팀**
  - G/L 계정: **343002 광고선전비(조사연구)**
  - 금액: VAT 제외 5,600만원 (JK/AM 각 2,800만원씩 분할, 26년 기준)
  - 첨부: 전년도 아웃풋 장표 + 엠브레인 견적서

---

## Phase 2: 구매품의 (연 1회, 예산품의 승인 후)

> 26년 구매품의 이미 완료 (작성자: 주호연). 참조 시 전자결재에서 검색.

### Step 2. 구매품의 작성
- ① 필요 서류: 엠브레인 견적서, 입찰생략요청서
- ② 입수처: 예산품의 승인 후 해당 페이지 → "하위문건 등록" 클릭
- ③ 시스템 입력값:
  - 문건 제목: 예산품의와 동일
  - 예산연동 유형: **대체**
  - 코스트센터/G/L 계정: 예산품의와 동일

---

## Phase 3: Concur 작성 (분기별, 용역 완료 후)

### [사전 준비] 필요 파일 수집

| 파일명 예시 | 입수처 |
|------------|--------|
| 예산품의 PDF | 잡코리아 포털 → MY문건 → 프린트 |
| 구매품의 PDF | 잡코리아 포털 → MY문건 → 프린트 |
| `250818_엠브레인_견적서.pdf` | 창준님 (엠브레인 CC 메일에서 다운) |
| `251016_엠브레인_거래명세표.pdf` | 창준님 (엠브레인 CC 메일에서 다운) |
| 조사완료 보고서 (1, 2부 분할) | 창준님 전달 |

> 파일명 날짜 기준: 견적서=견적일, 거래명세표=발행일, 조사완료 보고서=조사기간
> 세금명세서 PDF: 잡코리아 사업자번호 **1108134859** 입력 → 인쇄 → PDF 저장
> 파일 크기 10MB 초과 시: 내보내기 → PDF만들기 → 옵션 → 슬라이드 → 페이지 수 설정해 분할

### Step 3. 재무회계팀 송장 담당자 변경 요청
- ① 필요 서류: -
- ② 입수처: 재무회계팀 **박은미님**에게 요청
- ③ 시스템 입력값: Outlook으로 "송장이 지정됨" 메일 수신 → 메일 본문 "송장 처리 중 링크" 클릭

### Step 4. 연동 송장 작성 (Concur)
> 마감: 매달 익월 D+4 영업일 오전

- ① 필요 서류: 위 5종 PDF
- ② 입수처: concursolutions.com (이메일 링크로 로그인) → 송장 → 송장 관리자 탭
- ③ 시스템 입력값:

| 필드 | 값 |
|------|-----|
| A-01. 유형 | 일반 |
| A-02. 송장 이름 | `㈜마크로밀엠브레인-잡플레이스먼트 조사` |
| A-03. 증빙일 | 세금명세서 기준 날짜 (예: 2025-10-16) |
| A-04. 업무완료일 | 세금명세서 기준 날짜 |
| A-05. 전기일 | 세금명세서 기준 날짜 |
| A-06. 적요 | `엠브레인/251016/잡플레이스먼트서베이/오정윤` |
| B-01. 은행명 | (004) 국민은행 |
| B-02. 계좌번호 | (1000) 059250031658 |
| C-01. 통화 | KRW-Korea(Republic), Won |
| C-02. 총 합계 | 자동 기입 (연동 송장) |
| C-04. 부가세 합계 | 자동 기입 |
| D-01. 코스트센터 | (M2010) 잡코리아 마케팅팀 or (M3010) 알바몬 마케팅팀 (임의 선택 후 배분) |
| D-02. 지급 상태 | Not Paid |
| E-01. 세금코드 | (V1) 매입세금공제-10%세금계산서-일반 |
| E-02. 지급방법 | 계좌이체-원화 |
| E-03. AP 조정계정 | A.07 2101-02 매입채무-마케팅 |

### Step 5. 항목별 명세 요약 편집
- H-01. 경비 유형: **광고선전비(조사연구)** (연동 송장에서 Undefined → 변경 필수)
- H-02. 상세설명: `[25년 3분기] 잡코리아 잡플레이스먼트 조사` (자동 기입)
- H-03. 공급가액: 8,600,000 (자동 기입)
- H-05. 부가세: 860,000 (자동 기입)

### Step 6. 코스트센터 배분
- "배분" 탭 → 추가 → **(M2010) 잡코리아 마케팅팀 : (M3010) 알바몬 마케팅팀 = 50:50**
- 내부 오더: 미기입

### Step 7. 파일 첨부 & 제출
- 작업 → 이미지 업로드 → 예산품의, 구매품의, 거래명세표, 견적서, 조사완료보고서 업로드
- → 송장 제출
