# Topic: 신규 공급사 등록

## 개요
처음 거래하는 외부 업체를 잡코리아 공급사 시스템에 등록하는 절차.
등록 없이는 구매품의 및 Concur 처리가 불가함.

## 프로세스
1. 업체로부터 사업자등록증, 통장 사본 수령
2. 잡코리아 포탈 → 구매/공급사 관리 → 신규 공급사 등록 신청
3. 필수 입력 항목:
   - 업체명, 사업자번호
   - 대표자명, 업종/업태
   - 계좌 정보 (은행, 계좌번호)
   - 담당자 연락처
4. 총무팀 또는 재무회계팀 검토 후 등록 완료

## 주의사항
- 등록 소요 시간: 보통 1~3 영업일
- 긴급 시 재무회계팀 박은미님/왕윤형님에게 직접 연락

## 담당 연락처
- 재무회계팀 박은미님/왕윤형님 — 공급사 등록 처리
- 창준님 — 신규 업체 거래 승인
