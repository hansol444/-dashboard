# Topic: 엔코라인 통역 Concur 처리

## 개요
엔코라인 통역 서비스 이용 후 매월 Concur 송장 제출.
예산품의·구매품의는 **연간 일괄 처리 완료** (26년 계약 포함) → 평시 Concur 작성만 진행.

> **Concur 방식**: 엔코라인은 **세금명세서** 방식 → 재무회계팀에 담당자 변경 요청 후 연동 송장 처리
> (직접 새 송장 생성 X)

> ⚠️ 창준님 DM (2025-12-23): "엔코라인 예산품의 왜 올리신건가요? (안올리고 지금까지 해왔어서)"
> → 평시 예산품의 재신청 불필요. 예외 상황(계약 만료·예산 소진)에만 올릴 것.

---

## 예산품의 / 구매품의 (신규 계약·예산 소진 시에만)

> 현재: 26년 이미 진행 완료. 참고용 아래 기록.

- 파일 위치: 전략추진실 문서 > 98. 기타 행정처리 자료 폴더 (파일명 날짜만 수정 사용)
- 문의: 계약 담당자 남영현님 (총무팀), 문제 시 재무회계팀 김세진 팀장님

## 매월 Concur 작성 절차

### [사전 준비] 파일 수집

| 파일명 예시 | 기준 날짜 | 입수처 |
|------------|-----------|--------|
| `250526_엔코라인_견적서.pdf` | 견적일 | 창준님 전달 |
| `250526_엔코라인_세금명세서.pdf` | 세금명세서 발행일 | 창준님 전달 또는 직접 수령 |
| 예산품의 PDF | - | 잡코리아 포털 → MY문건 |
| 구매품의 PDF | - | 잡코리아 포털 → MY문건 |

> 세금명세서 PDF 저장: 잡코리아 사업자번호 **1108134859** 입력 → 인쇄 → PDF로 저장

### Step 1. 재무회계팀 송장 담당자 변경 요청
- ① 필요 서류: -
- ② 입수처: 재무회계팀 **박은미님** (또는 김세진 팀장님)에게 요청
- ③ 시스템: Outlook "송장이 지정됨" 메일 수신 → 본문 링크 클릭 → Concur 접속

### Step 2. 연동 송장 작성 (Concur)
> 마감: 매달 익월 D+4 영업일 오전

- ① 필요 서류: 견적서, 세금명세서, 예산품의, 구매품의 (4종 PDF)
- ② 입수처: concursolutions.com (이메일 링크로 로그인)
- ③ 시스템 입력값 (빨간 색 = 건마다 재작성):

| 필드 | 값 |
|------|-----|
| A-01. 유형 | 일반 |
| A-02. 송장 이름 | `㈜엔코라인-SEEK 통역` (통역 종류에 따라 변경) |
| A-03. 증빙일 | **세금명세서 기준** (예: 2025-05-26) |
| A-04. 업무완료일 | **업무 완료 기준** (예: 2025-05-26) |
| A-05. 전기일 | **세금명세서 기준** |
| A-06. 적요 | `엔코라인/250525/통역/윤경환/SEEK 컨퍼런스콜` |
| | ※ 재계약 후 첫 송장: `엔코라인/날짜/통역/담당자/항목/V60` 형식 |
| | ※ BOD 등 특수 건: "25년 BOD 진행을 위한 통역비용 발생" 등 상세 기입 (재무회계팀 요청) |
| B-01. 은행명 | (020) 우리은행 |
| B-02. 계좌번호 | (1000) 1005000926061 |
| C-01. 통화 | KRW-Korea(Republic), Won |
| C-02. 총 합계 | **견적서·세금계산서 확인** (예: 770,000) |
| C-04. 부가세 합계 | **견적서·세금계산서 확인** (예: 70,000) |
| D-01. 코스트센터 | **(H2200) 전략추진실** |
| D-02. 지급 상태 | Not Paid |
| D-03. 코멘트 | 공란 |
| E-01. 세금코드 | (V1) 매입세금공제-10%세금계산서-일반 |
| E-02. 지급방법 | 계좌이체-원화 |
| E-03. AP 조정계정 | A.06 2101-01 매입채무-일반 |
| F 부분 | 전체 공백 |

### Step 3. 항목별 명세 요약
- "다음에서 세금 지정" → "항목별 명세 요약" → 저장

| 필드 | 값 |
|------|-----|
| H-01. 경비 유형 | **지급수수료-기타** |
| H-02. 상세설명 | `엔코라인_5월26일 통역` (날짜 변경) |
| H-03. 공급가액 | 견적서 참고 (예: 700,000) |
| H-04. 세금코드 | (V1) 매입세금공제-10% 세금계산서-일반 |
| H-05. 부가세 | 세금명세서 참고 (예: 70,000) |

### Step 4. 파일 첨부 & 제출
- 이미지 업로드: 견적서, 세금명세서, 엔코라인 예산품의, 구매품의 첨부 → 업로드 → 저장
- 확인: 오른쪽 상단 "이미지 보기"
- 최종: "송장 제출"

### Step 5. 엔코라인 비용 History 업데이트
- "엔코라인 비용" Sheet → 엔코라인 비용 History 최신화 (견적서 참고)
