# Topic: 엔코라인 통역 컨커 처리

## 개요
엔코라인 통역 서비스 이용 후 Concur로 비용 처리.

## Concur 처리 절차
1. Concur 접속 → Create Expense Report
2. 핵심 필드:
   - **Report Name**: 엔코라인 통역 YYYYMM
   - **Business Purpose**: 통역 서비스
   - **Vendor**: 엔코라인
   - **Amount**: 청구 금액
3. 송장 첨부
4. 재무회계팀 담당자 변경 요청 (필요 시)
5. 제출

## 과거 이용 내역 (참고)
- 주로 외부 미팅, 컨퍼런스 통역 목적
- 건당 단가는 행사 규모/시간에 따라 다름

## 담당 연락처
- 재무회계팀 박은미님/왕윤형님 — Concur 담당자 변경, 계정 확인
- 창준님 — 이용 승인
