# Topic: 예산 101 (기초 개념)

## 예산품의 vs 구매품의
- **예산품의**: "이만큼 쓰겠다"는 사전 승인
- **구매품의**: "이 업체에서 산다"는 집행 승인
- 순서: 예산품의 승인 → 구매품의 작성 (하위문건) → Concur 처리

## 문서 분류 기준

### 예산품의
- 1,000만원 이하: [예산관리] 1.예산관리 - 3)예산배정(신규, 전용, 초과) - 1,000만원 이하
- 1,000만원 초과: [예산관리] 1.예산관리 - 3)예산배정(신규, 전용, 초과) - 1,000만원 초과

### 구매품의
- [구매] 해당 금액 구간 선택
- 500만원 이상: 입찰생략요청서 첨부 필요

## 코스트센터 / G/L 계정
- 업무별로 다름 → 모르면 **경영기획팀 임장식님** 또는 **예산관리 Chatbot** 문의
- 재무회계팀 박은미님/왕윤형님 — 세부 항목 및 계정 가이드

## 구매검토가 필요한 경우
- 산학협력, 외부 계약 등
- **총무팀 이민희님**에게 메일 (참조: 박현수님, 남영현님)

## 읽기 권한 문제
예산품의에 읽기 권한 없을 때 구매품의 등록 방법:
1. 경영기획팀에 열람권한 요청 후 하위문건 등록
2. 예산품의 PDF 첨부 + 비고란에 "예산품의 열람권한 없음에 따라 PDF 별첨" 코멘트

## 주의사항
- 예산품의 없는 구매품의 → 창준님께 먼저 보고
- 추가 예산 품의는 반려 확률 높음 → 명확한 증액 사유 필수

## 담당 연락처

## 서류 입수 경로 매핑

| 서류명 | 입수 유형 | 담당자/경로 | 구체적 실행 방법 |
|--------|----------|------------|----------------|
| 예산품의 (신규 작성) | B (사이트) | 잡코리아 포털 → 전자결재 → 문건등록 | 로그인 후 문건등록 → 예산품의 작성 |
| 구매품의 (신규 작성) | B (사이트) | 잡코리아 포털 → 전자결재 → 예산품의 하위문건 등록 | 예산품의 승인 후 해당 페이지에서 "하위문건 등록" 클릭 |
| 예산품의 PDF (기존 문건) | B (사이트) | 잡코리아 포털 MY문건 | 잡코리아 포털 → 전자결재 → MY문건 → 프린트 → PDF 저장 |
| GL 계정 확인 | C (Slack) | 경영기획팀 임장식님 or 예산관리 Chatbot | 슬랙 DM or 포털 예산관리 Chatbot |
| 입찰생략요청서 (500만원↑) | B (사이트) | 잡코리아 포털 자료실 → 문서서식 → 구매 및 계약 서식 | 다운로드 후 CSO님 서명 |
