# Topic: 나인하이어 지급 (에스크로 + 스톡옵션)

## 개요
나인하이어 관련 에스크로 수수료, 주식매매대금, 스톡옵션 보상 지급.
진행 순서: 에스크로 수수료 → 주식매매대금 → 스톡옵션

---

## 1. 에스크로 수수료
- **시기**: 매년 12월
- 절차: 예산품의 → Concur 처리
- 담당: 창준님 지시에 따라 진행

## 2. 주식매매대금
- 재직 여부 확인 대상:
  - 정승현
  - 이예린
  - 최돈민
  - 이경환
- 재직 중인 대상자에게만 지급
- 절차: 재직 확인 → 예산품의 → 지급 처리

## 3. 스톡옵션 보상
- 지급 대상:
  - 김재인
  - 안정태
  - 이정욱
- 절차: 예산품의 → 지급 처리

---

## 주의사항
- 에스크로 수수료는 연 1회 (12월) 고정
- 주식매매대금 지급 전 반드시 재직 여부 확인
- 모든 단계 창준님과 얼라인 후 진행

## 담당 연락처
- 창준님 — 전체 지급 지시 및 대상자 확인
- 재무회계팀 박은미님/왕윤형님 — 지급 처리
