# Topic: 나인하이어 지급 (에스크로·주식매매대금·스톡옵션)

## 개요
나인하이어 인수 관련 3종 지급 업무. **반드시 재직 여부 사전 확인**.
지급 순서: **에스크로 수수료 → 주식매매대금 → 스톡옵션** (에스크로가 선행되어야 함)

> ⚠️ 2026-03-03 기준: 잡코리아 → 웍스피어 사명 변경으로 계약서 변경 필요
> - 매수인 1) 인감 도장, 2) 법인명, 3) 인출지시서 법인명 변경
> - 변경합의서 6부 + 간인
> - 매도인 대표와 3인 위임장, 개인인감증명서 (매도인 4인)

---

## 1. 에스크로 수수료 (매년 12월 / 연 1회)

> 2023·2024·2025년 각 1,000만원씩 지급 완료

### Step 1. 예산품의 작성
- ① 필요 서류: 에스크로 계약서 PDF
- ② 입수처: 계약서 위치 → Teams 클라우드 `나인하이어 계약서 > 에스크로 계좌 내 주식 매매대금 입금` 폴더
- ③ 시스템: 잡코리아 포털 → 전자결재 → 문건등록 → 예산품의 작성 (첨부: 에스크로 계약서)

### Step 2. Concur 작성
- ① 필요 서류: 재무회계팀에서 발송한 메일 (송장 정보), 예산품의 PDF
- ② 입수처: 재무회계팀에서 온 메일 참고해서 작성
- ③ 시스템 입력값:
  - Step 1: 새 송장 생성 → 검색: 공급업체 / 시작문자 → `㈜하나은행 종로금융센터` 검색
  - Step 2: 컨커 작성 후 예산품의 이미지 첨부
  - 컨커 승인 후: 재무회계팀에서 보내준 **영수증 추가 업로드**

## 2. 주식매매대금 (26년 1월, 27년 1월)

> 반드시 재직 여부 확인 후 진행

### 재직 확인 대상자
| 대상자 | 비고 |
|--------|------|
| 정승현 | 매도인 대표 |
| 이예린 | - |
| 최돈민 | - |
| 이경환 | - |

### Step 1. 품의 작성
- ① 필요 서류: 이전 품의 참고 (개개인별 금액·일정 기 기입된 파일)
- ② 입수처: 기존 품의 복사 후 새로 등록. 대상자 재직 여부 먼저 확인.
- ③ 시스템: 잡코리아 포털 → 전자결재 → 문건등록

### Step 2. 인출지시서 작성
- ① 필요 서류: 인출지시서 (2차 인출지시서 참고해서 3·4차 작성)
- ② 입수처: Templates 위치 → `나인하이어 계약서 > 에스크로 계좌 내 주식 매매대금 입금`
- ③ 시스템 입력값: 각 개인별 금액·대상자·계좌·날짜 기입 → 매도인 대표 **정승현님**께 전달 (날인 + 인감증명서 요청)
  - 인출지시서 인감: 매도인 대표 정승현의 인감 4부로 갈음

### Step 3. Concur 작성
- ① 필요 서류: 품의 승인 완료
- ② 입수처: concursolutions.com
- ③ 시스템: 품의 승인 확인 후 Concur 작성 → 재무팀에서 은행 방문해 인출지시서 제출 → 완료
