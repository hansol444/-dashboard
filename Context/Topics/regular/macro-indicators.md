# Topic: Macro 지표 발굴 및 추가

## 개요
기존 Macro Analysis에 포함되지 않은 신규 고용/경제 지표를 발굴하고 엑셀에 추가하는 업무.
`update_macro.py` 자동화 범위 밖으로, 수동 판단이 필요함.

## 프로세스
1. 창준님으로부터 추가할 지표 방향 전달받기
2. KOSIS에서 해당 지표 데이터 탐색
3. 데이터 구조 확인 (연도별/월별/분기별, 산업 분류 기준)
4. Macro Analysis 엑셀에 새 시트 또는 기존 시트에 추가
5. `update_macro.py`에 새 시트 매핑 추가 (자동화 범위 확장 시)

## 주의사항
- 지표 추가 전 반드시 창준님과 방향 얼라인
- KOSIS 분류 기준이 기존 시트와 다를 경우 별도 시트 생성

## 담당 연락처

- KOSIS 문의: kosis.kr 고객센터
