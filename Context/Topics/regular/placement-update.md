# Topic: Placement Survey 업데이트 (쿼터/설문 변경)

## 개요
분기별 Placement Survey의 쿼터(표본 구성) 및 설문 문항을 변경하는 정기 업무.
엠브레인 문주원님과 협업하며 진행.

## 주요 작업 유형

### 쿼터 변경
- 연령/직종/경력 등 표본 구성 비율 조정
- 창준님으로부터 변경 방향 전달받아 엠브레인에 요청

### 설문 문항 변경
- 신규 문항 추가, 기존 문항 수정/삭제
- 문항 순서 변경

## 프로세스
1. 창준님으로부터 변경 사항 전달받기
2. 변경 내용 정리 (엑셀 또는 문서)
3. 엠브레인 문주원님에게 요청 메일/슬랙 발송
4. 엠브레인 확인 및 수정 완료 확인
5. 변경 내용 내부 기록

## 담당 연락처
- 엠브레인 문주원님 — Placement Survey 담당, 쿼터/설문 변경 요청 대상
- 창준님 — 변경 방향 결정
