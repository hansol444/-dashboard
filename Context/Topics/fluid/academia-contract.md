# Topic: 산학협력프로젝트 계약 & Concur

## 개요
대학교 학회(EGI, MCSA 등)와의 산학협력 프리랜서 계약 전 과정.
내부 Process: **예산품의 → 구매검토 → 구매품의 → 인장관리 → 계약 체결 → Concur 송장**

> ⚠️ 규정상 업무 시작 전 모든 절차 완료 필수.
> 프로세스 미완료 상태에서 업무 시작 시: 수신 이민희님·박현수님, 참조 김준수님·이상인님으로 사전 승인 이메일 발송 필요.

---

## Phase 1: 계약 체결 프로세스 (5단계)

### Step 1. 예산품의 작성
- ① 필요 서류: 학회 견적서 (또는 창준님 제공 내용)
- ② 입수처: 잡코리아 포털 → 전자결재 → 문건등록
- ③ 시스템 입력값:
  - 코스트센터: **(H2200) 전략추진실**
  - G/L 계정: **지급수수료-사업소득** (프리랜서)
  - 문건분류: 금액에 따라 `1,000만원 이하` or `1,000만원 초과`
  - 내용: "배경 및 목적, 필요 예산 및 방식, 예산의 기대효과 순으로 작성"
    - 필요 예산 및 방식에: `"2026년 사업계획 상에 1800만원 예산이 배정되었고, 해당 예산을 사용하겠습니다"` 문구 포함 (창준님 요청)
  - 첨부: 견적서

### Step 2. 구매검토 요청 (이메일)
- ① 필요 서류: 계약서 초안 (계약명·금액·기간·범위 명시)
- ② 입수처: 총무팀
  - 수신: 이민희님 (총무팀)
  - 참조: 박현수님, 남영현님
- ③ 시스템: 상대방측 연락처(이메일주소) 함께 전달 → 총무팀이 계약 당사자에게 이메일 발송 → 회신 받으면 구매검토 완료 통보

> 계약금액 **500만원 이상** → 입찰생략요청서 추가 제출
> - 입찰생략요청서 다운: 포털 → 자료실 → 문서서식 → 구매 및 계약 서식
> - 생략 사유: ①대학생 산학협력은 학회 제안 방식이 일반적, ②해당 학회 우수성 서술
> - CSO님 서명: 지면 서명 후 스캔 or 예외 시 전자서명(PDF 파일에 펜 서명/서명 이미지)

### Step 3. 구매품의 작성
- ① 필요 서류: 견적서, 구매검토 완료된 계약서, 입찰생략요청서 (500만원 이상 시)
- ② 입수처: 예산품의 페이지 → "하위문건 등록" 클릭
- ③ 시스템 입력값:
  - 문건 제목: 예산품의와 동일
  - 문건분류: `[구매] 2) 100만원 이상 ~ 1,000만원 미만`
  - 첨부: 구매검토 완료 계약서 (입찰생략요청서 있으면 함께 압축)
  - 읽기권한 없을 경우 → 예산품의 PDF 첨부 후 비고란에 `"예산품의 열람권한 없음에 따라 예산품의 PDF 별첨"` 코멘트

### Step 4. 인장관리 (CSO 날인)
- ① 필요 서류: 구매품의 승인 완료
- ② 입수처:
  - 사용인감: MY메뉴 → 인장관리 → 사용인감신청
  - 법인인감: Support-법인인감날인 슬랙 채널 신청
- ③ 시스템: -

### Step 5. 계약 체결
- ① 필요 서류: 날인 완료 계약서
- ② 입수처: 총무팀 **남영현님**과 최종 계약서 체결
- ③ 시스템: 체결 완료 후 계약서 사본 Teams 폴더에 보관

## Phase 2: 신규 공급업체 등록 (처음 거래 학회/업체)

> 학회는 비영리 단체 → **학회 대표자 개인**으로 등록

- ① 필요 서류: 대표자 신분증 사본, 계좌사본
- ② 입수처: 인트라넷 → 전자결재 → 신규공급업체 등록
- ③ 시스템: 등록 승인 완료 후 Concur 새 송장 "A.01 유형 = 원천세" 로 변경, 검색에서 "공급업체 + 포함문자 + 대표자 이름"

---

## Phase 3: Concur 송장 작성 (프로젝트 종료 후)

### [사전 준비] 파일 수집
- ① 필요 서류 (3종): 예산품의 PDF, 구매품의 PDF, 계약서 PDF
- ② 입수처: 잡코리아 포털 MY문건 → 프린트 → PDF 저장

### Step 6. Concur 새 송장 작성 (원천세 유형)
> 마감: 매달 익월 D+4 영업일 오전

- ① 필요 서류: 위 3종 + 프로젝트 결과물 (중간보고 있으면 포함, 최종보고 필수)
- ② 입수처: concursolutions.com (이메일 링크로 로그인) → 송장 → 새 송장 생성
- ③ 시스템 입력값:

| 필드 | 값 |
|------|-----|
| A-01. 유형 | **원천세** |
| A-02. 송장 이름 | `홍길동(학회 대표자)-MCSA 산학협력프로젝트 용역 수주 정산` |
| | ※ MCSA의 경우 공급업체 검색 시 "서울대 산학협력단"으로 검색 |
| A-03. 증빙일 | **업무 완료된 기준** (예: 2025-06-05) |
| A-04. 업무완료일 | 업무 완료된 기준 |
| A-05. 전기일 | 업무 완료된 기준 |
| A-06. 적요 | `홍길동(학회 대표자)/250605/서울대 MCSA 산학협력프로젝트/강하니/V30` |
| B-01. 은행명 | 자동 선택 |
| B-02. 계좌번호 | 자동 선택 |
| C-01. 통화 | KRW-Korea(Republic), Won |
| C-02. 총 합계 | 계약서 기준 |
| D-01. 코스트센터 | **(H2200) 전략기획실** |
| D-02. 지급 상태 | Not Paid |
| E-01. 원천세율 | **1. 사업소득(거주자) 3%** |
| E-02. 원천징수여부 | Yes |
| E-03. 지급방법 | 계좌이체-원화 |

### Step 7. 항목별 명세 요약
| 필드 | 값 |
|------|-----|
| H-01. 경비 유형 | **지급수수료-사업소득** |
| H-02. 상세설명 | `MCSA 산학협력프로젝트 용역 6주 고용에 따른 금액 정산` |
| H-03. 공급가액 | 계약서 참고 |

### Step 8. 파일 첨부 & 제출
- 이미지 업로드: 예산품의, 구매품의, 계약서, 프로젝트 결과물 첨부 → 저장
- 확인: 오른쪽 상단 "송장 보기"
- 최종: "송장 제출"

| 서류명 | 입수 유형 | 담당자/경로 | 구체적 실행 방법 |
|--------|----------|------------|----------------|
| 견적서 (학회) | A (사람) | 창준님 또는 학회 직접 | 창준님께 "산학협력 견적서 공유해 주실 수 있을까요?" 슬랙 DM |
| 예산품의 PDF | B (사이트) | 잡코리아 포털 MY문건 | 잡코리아 포털 → 전자결재 → MY문건 → 프린트 → PDF 저장 |
| 구매품의 PDF | B (사이트) | 잡코리아 포털 MY문건 | 동상 |
| 입찰생략요청서 (500만원↑) | B (사이트) | 포털 자료실 → 문서서식 → 구매 및 계약 서식 | 다운로드 후 작성. CSO님 서명 필요 |

| 프로젝트 결과물 | A (사람) | 학회 (EGI/MCSA 등) | Slack 산학협력 채널 or 이메일로 수령 |
